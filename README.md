# Simple calculator:

This is very simple calculator which is taking 2 operand and a operator as input and giving its solution.

To run the code:
g++ filename.cpp -o name
.\name

Input:
Code is taking 2 operand and a operator as input and it is taking in sequence, 1st operand, operator, 2nd operand
operands are integers.

Output:
Output is simply the mathematical calculation of the input.

Example:
first operand: 12
operator: +
second operand: 8
20
